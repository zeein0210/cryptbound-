# 🃏 수명 베팅 카드 게임 (Life-Betting Card Game)

동물 카드와 본인의 '수명'을 걸고 싸우는 전략 카드 게임입니다.

## 🐾 카드 정보
- **나비 (S):** 10점 | **두루미 (A):** 8점 | **제비 (B):** 6점 | **사슴 (C):** 4점 | **멧돼지 (D):** 2점

## 🔄 5각 상성 (승리 +3점 / 패배 -1점)
- 나비 ➔ 제비 ➔ 멧돼지 ➔ 두루미 ➔ 사슴 ➔ 나비

## ✨ 카드별 특수 능력
- **나비:** 이전 사용한 패 1장 재사용 가능.
- **두루미:** 패배 시 손실 수명 절반 / 승리 시 2배 획득.
- **제비:** 상대 배팅 수명을 미리 엿본다.
- **사슴:** 상대 패의 중복 여부를 간파한다.
- **멧돼지:** 모든 특수 효과 무효화 및 수치 대결 강제.

## ⚖️ 공식
`총점 = 카드 등급 점수 + 베팅 수명(1년=1점) + 상성 보너스`
